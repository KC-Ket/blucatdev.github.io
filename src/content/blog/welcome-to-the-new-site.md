## New look, who this?

Welcome to the redesigned Blu Cat Dev website!

### What

This website is now powered by Astro and hosted on GitHub Pages. It uses Cloudflare to
allow remote updates. It uses Zapier to push RSS feeds (e.g. new blog posts to Discord).
I used Claude Code to help be rebuild it, which feels like cheating as a software engineer,
but "web" languages aren't my forte so I appreciate the help.
JavaScript and I have a violent past.

### Why

My goals were to modernise my website, align the core design with other studios, and simplify the ways
I could add and update content. I can now make changes locally on my computer,
or on the go. I have also added poems to the writing section to encourage myself to write more.
I am a very creative person (not necessarily talented, though), so I wanted more control in
what and how I shared my creations with the world.
